# film-navigation-taxonomy-lab
本仓库围绕影视类网站在导航分类体系设计与页面逻辑方面的公开资料进行整理，通过分析分类标签划分方式、内链分布逻辑以及关键词排布策略，形成系统化研究档案。项目不包含影视播放资源或下载链接，仅对公开页面元素进行记录说明。通过长期整理不同站点的结构特征，可用于研究页面抓取路径与内容组织模式。所有资料来源于可公开访问页面，仅供信息索引与结构研究使用，不构成商业宣传或背书行为。
